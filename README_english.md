[SIMANFOR](https://www.simanfor.es) is a Decision Support System that allows the simulation of forest management alternatives.

In this repository, you will find content related to the SIMANFOR website and how to use it.

---

## 📜 Content

* 📂 **logos**: Folder containing various logos used in these explanatory documents (README.md) of the [SIMANFOR GitHub repository](https://github.com/simanfor)
* 📖 🇪🇸 **SIMANFOR_web.pdf**: Guide to using the SIMANFOR website in Spanish
* 📖 🇬🇧 **english/SIMANFOR_web.pdf**: Guide to using the SIMANFOR website in English
